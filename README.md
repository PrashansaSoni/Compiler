#  Simple Compiler Project

This project is a basic **compiler** that performs essential compilation steps such as lexical analysis, parsing, and syntax validation. It is designed to help understand how compilers work internally.

---

##  Features
- Reads and processes input code  
- Performs **tokenization** (lexical analysis)  
- Identifies keywords, identifiers, operators, and numbers  
- Checks syntax based on defined grammar  
- Shows clear error messages for invalid code  
- Displays final result: **Valid Code / Invalid Code**
